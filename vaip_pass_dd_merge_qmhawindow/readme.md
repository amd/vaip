# DD flow QMHQWINDOW OP


## config

to enable this pass, we need to add the following config into `vaip_config.json`

```json
{
   "name": "vaip_pass_dd_merge_qmhawindow",
   "plugin": "vaip-pass_dd_merge_qmhawindow",
   "disabled": false
}
```
